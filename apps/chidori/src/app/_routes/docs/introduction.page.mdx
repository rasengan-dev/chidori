# @rasenganjs/kage-demo

`@rasenganjs/kage-demo` is an **official demonstration package** of the Rasengan.js ecosystem.

Its goal is to **showcase the core features of Rasengan.js in real-world scenarios**, acting as both:

* a **learning tool** for newcomers
* a **reference implementation** for experienced developers


<Note title="Note">
Think of it as the *“Kage”* of Rasengan.js: it demonstrates mastery of the framework’s techniques.
</Note>

## ✨ Purpose

This package is designed to:

* Demonstrate **Rasengan.js rendering modes**
* Showcase **routing, layouts, and metadata**
* Illustrate **state management with Kurama**
* Provide **realistic examples** without boilerplate complexity
* Serve as a **playground and inspiration** for new projects

It is **not meant for production use**, but as a **reference and exploration tool**.

## 🐺 Kurama Integration (State Management)

`@rasenganjs/kage-demo` includes **simple and advanced examples** using
`@rasenganjs/kurama`.

### Store Definition

```ts title="counter.ts" showLineNumbers
import { createStore } from '@rasenganjs/kurama';

export const counterStore = createStore({
  state: { count: 0 },
  actions: {
    increment(state) {
      state.count++;
    }
  }
});
```

### Usage in Components

```tsx title="src/app/counter.tsx" showLineNumbers {2}
import { useStore } from '@rasenganjs/kurama';
import { counterStore } from './store';

export function Counter() {
  const { state, actions } = useStore(counterStore);

  return (
    <button onClick={actions.increment}>
      Count: {state.count}
    </button>
  );
}
```

This demonstrates:

* Global state
* Predictable updates
* Framework-native integration
