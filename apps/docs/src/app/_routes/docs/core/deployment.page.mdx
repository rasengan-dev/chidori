# Deployment

To deploy a `Chidori` website you have to follow exactly the deployment guide of `Rasengan.js`.

Rasengan.js provides you a great way to deploy apps via its `adapters`. For your specific need, select the adapter you want to use from this list:

- <Link to="https://rasengan.dev/packages/vercel" target="_blank">Vercel</Link>
- <Link to="https://rasengan.dev/packages/netlify" target="_blank">Netlify</Link>
- <Link to="https://rasengan.dev/packages/serve" target="_blank">Node</Link>

<Note title="Info">
`Node` adapter is special for hosting apps on a VPS
</Note>
